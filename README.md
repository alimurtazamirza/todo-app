# Todo Siri App (Implementation Started)

This repository now contains an initial SwiftUI implementation for an iPhone to-do app with Siri support.

## What is implemented

- Local to-do storage with persistence
- SwiftUI list/add UI
- Siri/App Intents command to add tasks by voice

## Next steps

- Create an Xcode iOS app target and add files from `ios/TodoSiriApp/`
- Enable Siri capability and App Intents
- Optionally move persistence to an App Group shared store for intent extension compatibility
